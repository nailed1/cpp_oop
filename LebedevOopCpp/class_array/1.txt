    График 1 (слева): Временная сложность
     - Синяя линия — экспериментальное время выполнения алгоритма для разных N
     - Красная пунктирная — теоретическая кривая O(N²)
     - Обе оси в логарифмическом масштабе
     - Показывает, что реализация с Array имеет квадратичную сложность

    График 2 (справа): Проверка правильности
     - Зелёная линия — значения по точной формуле для k=2: J(N,2) = 2*(N - 2^floor(log₂(N))) + 1
     - Красная линия — результаты вашего алгоритма на Array
     - Чёрная пунктирная — линия идеального совпадения y=x
     - Если точки лежат на этой линии — расчёты верны

    Дополнительно
    После построения выводится таблица со сравнением расчётных значений и формулы, где ✓ означает совпадение, ✗ — расхождение.

    Итог: скрипт проверяет, что ваша реализация работает правильно (совпадение с формулой) и имеет ожидаемую производительность O(N²).
